Les structures conditionnelle ternaire

tougoudou, les ternaires sont une autre façon d'écrire les conditions en utilisant une autre syntaxe basée sur l'opérateur ternaire ?:.

1- Ecriture

On peut écrire la condition sur une seul ligne

Ex : document.getElementById('p4').innerHTML = y >= 10 ? 'y est inférieur ou égal à 10' : 'y est inférieur à 10';

ls ternaires se présentent sous la forme suivante : test ? code à exécuter si true : code à éxecuter sinon
