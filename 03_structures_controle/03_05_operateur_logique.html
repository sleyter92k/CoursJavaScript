Les opérateurs logiques

Ce type d'opérateur va nous permettre d'effecteur plusieurs comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test

1- Présentation et utilisation des opérateurs logiques

Opérateur logiques
Nom Symbole Description
AND (ET) && renvoi true si toutes les comparaisons est vraie, oufalse sinon.
