Javascript - IV/ 03 - Les valeurs de retour

Return

Définition

En JS, il existe deux espaces de portée différents : l'espace global et l'espace local.

  • L'espace global désigne l'entièreté d'un script, à l'exception de l'intérieur des fonctions.
  • L'espace local désigne, à l'inverse, l'espace dans une fonction.
